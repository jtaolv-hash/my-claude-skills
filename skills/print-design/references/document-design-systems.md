# Document Design Systems

Pre-built visual systems for common document types. Use these as starting points —
adjust based on the user's brand or specific context.

---

## Investor / Pitch Deck

**Palette**:
  - Background: `#0a0c14` (near-black navy)
  - Surface: `#12151f`
  - Accent: `#4f9eff` (electric blue) or `#60e080` (growth green)
  - Text primary: `#f0f4ff`
  - Text muted: `rgba(240,244,255,0.5)`

**Typography**:
  - Display: Syne 700 or DM Serif Display (for hero numbers/statements)
  - Body: DM Sans 400/500
  - Data: DM Mono

**Layout rules**:
  - Slide margins: 80px all sides
  - Section divider slides: full-bleed color with one large number or word
  - Chart slides: chart takes 60% of slide, insight takes 40%
  - Max body copy: 40 words per slide

**Visual language**: Clean geometric accents (horizontal rules, corner marks). No photography unless product shots. Data visualizations use the accent color only.

## Brand Guidelines Document

**Palette**:
  - Background: `#ffffff`
  - Section headers: Brand primary color (user-defined)
  - Body text: `#1a1a1a`
  - Dividers: `#e8e8e8`

**Typography**:
  - Use the brand's own typefaces (the whole point is demonstrating the system)
  - If brand type is unavailable: Georgia (serif) or system-ui (sans)

**Layout rules**:
  - Cover: full-bleed brand color, logo centered or offset
  - Section openers: large section number + title, minimal
  - Example spreads: 60% example, 40% specification
  - Always show DO / DON'T pairs with visual examples

**Structure**:
  1. Brand Story (1–2 pages)
  2. Logo System (mark, wordmark, lockups, spacing rules)
  3. Color System (swatches, values, usage guidance)
  4. Typography System (specimens, scale, pairing rules)
  5. Photography / Illustration Style
  6. Applications (business card, letterhead, social, web)

## Annual Report / Corporate Publication

**Palette**:
  - Background: `#fafaf8` (warm white)
  - Accent: Corporate primary color
  - Data positive: `#16a34a`
  - Data negative: `#dc2626`
  - Text: `#111827`

**Typography**:
  - Display: A refined serif (Libre Baskerville, Playfair Display)
  - Body: A clean sans (Source Sans, DM Sans)
  - Data: Any monospaced (DM Mono, IBM Plex Mono)

**Layout rules**:
  - 12-column grid, 20px gutters
  - Pull quotes: 2 columns wide, large type, brand color
  - Financial tables: always full width, alternating row shading
  - Photography: full-bleed chapter openers, inset for supporting images

**Structure**:
  1. Letter from Leadership
  2. Year in Review (visual highlights)
  3. Financial Performance
  4. Business Segments
  5. Sustainability / ESG
  6. Financial Statements

## White Paper / Research Report

**Palette**:
  - Background: `#ffffff`
  - Accent: Single brand color (used sparingly for callouts only)
  - Text: `#1f2937`
  - Callout background: `#f3f4f6`

**Typography**:
  - Body: Georgia or a legible serif — research reads better in serif
  - Headings: A strong sans (Syne, DM Sans Bold)
  - Data/code: DM Mono

**Layout rules**:
  - Single column for body (max 70 characters per line — readable line length)
  - Sidebar column (20% width) for callout stats, definitions, pull quotes
  - All charts within the text column, never full-bleed
  - Footnotes, not endnotes — keep citations close to claims

## Packaging (Label / Box Front Panel)

**Palette**: Defined by brand (user provides or Claude proposes)

**Hierarchy** (top to bottom, front panel):
  1. Brand name — dominant
  2. Product variant / descriptor
  3. Key claim (one short phrase)
  4. Visual element (illustration, pattern, or product photography)
  5. Net weight / volume (bottom, required)

**Layout rules**:
  - Bleed: 3mm on all sides for print production
  - Safe zone: All text 5mm from trim edge minimum
  - Contrast: Text must be legible at arm's length
  - Color: Spot colors preferred for premium; CMYK for standard print runs

**Typography**: Display weight for brand name. Regular weight for product name.
Light weight for descriptors and legal. Never more than 2 typefaces on packaging.
