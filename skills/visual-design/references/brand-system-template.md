# Brand Identity Delivery Format

Standard structure for brand identity deliverables from the visual-design skill.

---

## Brand Identity Package

### 1. Logo System
```
Primary logo:     Full lockup (mark + wordmark)
Secondary logo:   Wordmark only
Icon/Mark:        Symbol only (for favicon, app icon)
Reversed:         White version for dark backgrounds
```
**File formats**: SVG (vector master) + PNG (screen use)
**Minimum sizes**: Icon no smaller than 24px; full logo no smaller than 120px wide

### 2. Color System
```
Primary:    #XXXXXX   "Name"   — Main brand expression
Secondary:  #XXXXXX   "Name"   — Supporting accent
Neutral 1:  #XXXXXX   "Name"   — Text / dark backgrounds
Neutral 2:  #XXXXXX   "Name"   — Light surfaces / backgrounds
```
For each color, specify:
- HEX (digital)
- RGB (screen)
- CMYK (print)
- Pantone (physical production, if applicable)

**Accessibility**: Primary on Neutral 2 must pass WCAG AA (4.5:1 contrast minimum)

### 3. Typography System
```
Display face:   [Typeface Name], [Weight(s)]
  Use for:      Headlines, large callouts, brand moments
  Sizes:        Heading 1 / Heading 2 / Heading 3

Body face:      [Typeface Name], [Weight(s)]
  Use for:      Body copy, UI text, captions
  Sizes:        Body / Small / Caption
```
**Line height**: Body = 1.5–1.65x. Display = 0.9–1.1x.
**Letter spacing**: Display = -0.01 to -0.03em. Body = 0 to +0.02em.

### 4. Logo Usage Rules
```
✓ Do:
  - Use on approved background colors only
  - Maintain minimum clear space (= height of the "X" in the logo)
  - Scale proportionally

✗ Don't:
  - Stretch, distort, or rotate
  - Apply drop shadows or effects
  - Place on busy photographic backgrounds
  - Use colors outside the approved system
```

### 5. Visual Language Notes
- Photography style (if applicable): [e.g., "warm, natural light; no stock-photo compositions"]
- Graphic elements: [e.g., "geometric rule lines at 1px; no decorative borders"]
- Tone: [e.g., "confident but not aggressive; refined but not cold"]

---

## Delivery Checklist

- [ ] Logo SVG (all variants)
- [ ] Color palette (HEX + RGB values)
- [ ] Typography spec (font names, weights, sizes)
- [ ] 1-page usage guide (PDF)
- [ ] Optional: Mockups showing logo in context
