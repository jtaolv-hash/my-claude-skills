# Skill: FPA Reporting

## Purpose
Support FPA and reporting workflows in luxury hospitality and PE contexts.

## User Context
- 10+ years FPA, luxury hospitality
- Tools: Snowflake, SQL, BI tools, NetSuite
- PE context: portfolio performance, asset-level reporting
- Familiar with: variance analysis, forecasting, scenario modeling

## Skill Rules
- Use FPA-native language: variance, bridge, waterfall, runway, EBITDA
- Default to structured outputs: tables, bridges, scenario comparisons
- Show logic before formula when building models
- Tie KPIs to hospitality benchmarks: RevPAR, GOP%, NOP
- Flag data assumptions explicitly

## Output Format
- Lead with: What changed | Why | So what
- Use variance bridge structure for financial analysis
- Tables preferred over prose for numerical output
- Scenario format: Base | Upside | Downside
